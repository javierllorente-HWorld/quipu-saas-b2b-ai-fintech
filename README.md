# Quipu

SaaS B2B fintech para gestión financiera de pymes en LATAM.

## Qué es

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Requisitos

- Node.js 20+

## Correr en local

Instalar dependencias:

```bash
npm install
```

Levantar en modo desarrollo:

```bash
npm run dev
```

Abrir `http://localhost:3000`.
