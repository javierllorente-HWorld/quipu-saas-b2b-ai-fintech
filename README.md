# Quipu

SaaS B2B fintech con IA para gestión financiera de pymes en LATAM.

## Qué es

Quipu es una web app para centralizar caja, cobros, pagos, tesorería, reportes y análisis financiero en un solo lugar.

## Para quién es

Dueños, gerentes generales, equipos financieros y altos mandos de pymes que necesitan visibilidad clara sobre la salud financiera del negocio.

## Problema que resuelve

Las pymes suelen tener información financiera dispersa entre bancos, planillas, facturas y sistemas separados. Esto dificulta saber cuánta caja tienen, qué deben cobrar, qué deben pagar y cómo viene el negocio.

## Propuesta de valor

Quipu permite tomar mejores decisiones financieras con datos claros, reportes simples, alertas accionables y soporte de IA.

## Alcance del MVP

- Login
- Dashboard inicial
- Caja
- Cobros
- Pagos
- Tesorería
- Reportes
- Copiloto IA
- Visualización de KPIs, tablas y gráficos
- Alertas y acciones sugeridas
